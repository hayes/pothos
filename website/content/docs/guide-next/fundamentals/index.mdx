The Fundamentals section covers each piece of the Pothos API in the order you'll naturally encounter it: the schema builder, then object types, fields, resolvers, context, arguments and inputs, the three root types, and finally interfaces, unions, enums, and scalars.

You don't need to read these top-to-bottom — each page stands alone. But the order matches how concepts depend on each other, so reading sequentially won't leave you with forward references.
