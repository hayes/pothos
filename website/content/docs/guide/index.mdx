The guide is in three parts:

- **[Getting started](./getting-started)** — install Pothos, build a first schema, wire it into a server. Start here if you're new.
- **[Fundamentals](./fundamentals)** — per-concept reference for every core building block, in dependency order.
- **[Patterns](./patterns)** — cookbook-style recipes for the recurring shapes in a Pothos schema.

The pages assume reading order within each section, but every page stands alone. Drop in at whichever one you need; cross-links carry you to anything they depend on.

<Callout type="info">
  Looking for plugin docs? Each plugin lives under [Plugins](../plugins). The guide here is core-only — plugins extend the core API without changing the patterns covered in these pages.
</Callout>
