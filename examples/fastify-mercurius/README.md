# Pothos + Fastify + Mercurius example

Minimal example showing how to serve a [Pothos](https://pothos-graphql.dev) schema
through [Fastify](https://fastify.dev) using the [Mercurius](https://mercurius.dev)
GraphQL adapter.

## Running

```sh
pnpm install
pnpm start
```

Then open [http://localhost:3000/graphiql](http://localhost:3000/graphiql).
